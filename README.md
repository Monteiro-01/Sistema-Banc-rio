# 🏦 Sistema Bancário

Este projeto é uma implementação simples de um **sistema bancário** em Python, criado para fins de estudo e prática de programação.  
O objetivo é simular operações básicas de um banco, como criação de contas, depósitos, saques e transferências.

---

## 🚀 Funcionalidades
- Criar contas bancárias
- Realizar depósitos
- Efetuar saques
- Transferir valores entre contas
- Consultar saldo

---

## 🛠️ Tecnologias Utilizadas
- **Python 3.x**
